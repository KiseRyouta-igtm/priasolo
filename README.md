## Deskripsi Repository

Repository ini dibuat sebagai latihan kolaborasi menggunakan Git dan GitHub pada mata kuliah Software Development.

Project ini mensimulasikan workflow pengembangan software dalam tim menggunakan GitHub seperti:

- Clone repository
- Membuat branch
- Commit perubahan
- Push branch
- Pull Request
- Merge Pull Request
