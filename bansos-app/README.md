Bansos App  
Deskripsi  
Bansos App adalah aplikasi sederhana yang dibuat untuk membantu proses pendataan bantuan sosial sekaligus memonitor status penyaluran bantuan secara lebih terstruktur.

Data yang digunakan berasal dari input pengguna yang kemudian diproses oleh sistem untuk memastikan data penerima bantuan valid dan tepat sasaran. Selain itu, aplikasi ini memungkinkan pengguna untuk melihat status bantuan apakah sudah disalurkan atau belum.

Fitur  
Menginput data penerima bantuan  
Validasi data penerima bantuan  
Monitoring status penyaluran bantuan  
Pengelolaan jenis bantuan (BLT, sembako, dll)  
Menampilkan data penerima bantuan  

Cara Kerja  
Pertama, pengguna memasukkan data penerima bantuan ke dalam sistem. Setelah itu, data akan diproses dan divalidasi. Selanjutnya, data disimpan dan dapat ditampilkan kembali untuk melihat status bantuan apakah sudah disalurkan atau belum.

Tools yang digunakan  
Python  
Flask  
SQLite  
Bootstrap  

Penutup  
Aplikasi ini dibuat untuk membantu proses pendataan bantuan sosial agar lebih efektif, akurat, dan mudah dikelola.
